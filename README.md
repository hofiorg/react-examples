# react_examples &middot; ![node_js workflow](https://github.com/hofiorg/react_examples/actions/workflows/node.js.yml/badge.svg)

## Installation

```sh
git clone https://github.com/hofiorg/react_examples.git
cd react_examples
npm install
```

## Start

```sh
npm start
```

## Test

```sh
npm test
```

## Build

```sh
npm run build
```
